# Changelog

All notable changes to Extract-STFS are documented in this file.

## [1.0.0] - 2026-03-05

### Added
- STFS container extractor for Xbox 360 LIVE, PIRS, and CON package types
- Block addressing algorithm supporting L0, L1, and L2 hash table layouts
- Female (shift=0) and Male (shift=1) block separation modes
- Consecutive and chained block allocation support
- `-ListOnly` mode for viewing package contents without extracting
- Package metadata display (type, header size, display name, file listing)
